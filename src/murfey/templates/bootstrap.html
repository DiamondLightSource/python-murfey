How to bootstrap pip without pip being present on the system:

      curl -sOL https://{{ hostname }}:{{ port }}/packages/py3/p/pip/pip-21.3.1-py3-none-any.whl
      python pip-21.3.1-py3-none-any.whl/pip install pip
